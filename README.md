# Дубкова Вероника Александровна 4ИСИП-422

## Приложение "Сервис частных объявлений"

---

## Описание проекта

Данное приложение реализовано на платформе **WPF (.NET Framework)** с использованием **Entity Framework** для взаимодействия с базой данных **SQL Server**.  
Оно предназначено для **размещения, поиска и управления частными объявлениями** о купле-продаже товаров, аренде недвижимости и предоставлении услуг.  
В приложении реализовано **разделение ролей** - разграничение прав доступа для гостей и авторизованных пользователей.

---

## Основной функционал

* **Для гостя:** 
  - просмотр списка всех активных объявлений;
  - поиск объявлений по ключевым словам, а также фильтрация по городу, категории, типу и статусу объявления;
  - возможность авторизации в системе.

* **Для авторизованного пользователя:**
  - добавление, редактирование и удаление собственных объявлений;
  - изменение статуса объявлений ("Активно"/"Завершено");
  - просмотр завершённых объявлений с подсчетом полученной прибыли.

* **Интуитивный интерфейс** 
  Приложение выполнено в едином согласованном стиле, реализованы подсказки в полях ввода, обработка ошибок и плавная навигация между страницами.
